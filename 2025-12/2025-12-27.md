### **1. 변수와 자료형**

**변수 (Variable)**

- 데이터를 저장하는 메모리 공간의 **이름** (예: `name = "Python"`)
- **동적 타이핑:** 자료형을 미리 선언할 필요 없이, 값에 따라 자동 결정
- **규칙:** 영문/숫자/`_` 사용, **숫자 시작 불가**, 대소문자 구분, 예약어 금지

**기본 자료형 (Scalar Types)**

- **`int` (정수):** 소수점 없는 숫자, 크기 무제한
- **`float` (실수):** 소수점 있는 숫자 (`3.14`)
- **`str` (문자열):** 글자들의 집합 (`"Hello"`), **변경 불가(Immutable)**
- **`bool` (논리):** `True` / `False`
- **`None`:** 값이 없음

```
import sys
sys.stdout.reconfigure(encoding='utf-8')

# 정수, 실수, 문자열, 불린
num = 10
pi = 3.14
text = "Python"
check = True

# 타입 확인
print(type(num))   # <class 'int'>
print(type(text))  # <class 'str'>
  ```

**컬렉션 자료형 (Collection Types)**

| **자료형** | **기호** | **순서** | **중복** | **변경** | **특징** |
| --- | --- | --- | --- | --- | --- |
| **List** | `[ ]` | O | O | **가능** | 데이터 추가/삭제가 잦을 때 |
| **Tuple** | `( )` | O | O | **불가** | 변하면 안 되는 고정 값 |
| **Dict** | `{ k:v }` | X | Key X | **가능** | Key-Value 쌍으로 관리 |
| **Set** | `{ }` | X | X | **가능** | **중복 제거**, 집합 연산 |


```
import sys
sys.stdout.reconfigure(encoding='utf-8')

# 리스트 (순서 O, 변경 O)
fruits = ["apple", "banana"]
fruits.append("orange") # 추가
fruits[0] = "grape"     # 수정
print(f"과일 목록: {fruits}")

# 딕셔너리 (Key-Value)
score = {"math": 90, "eng": 80}
score["math"] = 100     # 값 수정
score["sci"] = 95       # 새 키-값 추가
print(f"성적표: {score}")
```

### **2. 조건문과 반복문**

**조건문 (`if`)**

- 조건의 **참(True) / 거짓(False)**에 따라 코드 실행
- **구조:** `if` → `elif` (옵션) → `else` (옵션)
- **False 조건:** `0`, `None`, 비어있는 컬렉션(`[]`, `""`)

**반복문 (`for`, `while`)**

- **`for` 문:** 정해진 횟수나 데이터(리스트 등)만큼 반복
    - `range(시작, 끝, 간격)` 활용 가능 (끝 번호 제외)
- **`while` 문:** 조건이 참인 동안 계속 반복 (무한 루프 주의)
- **제어:** `break` (즉시 종료), `continue` (다음 순서로 건너뛰기)

### **3. 함수 정의와 호출**

**구조**

Python

`def 함수명(매개변수):
    실행코드
    return 반환값`

- **`def`:** 함수 정의 키워드
- **매개변수(Parameter):** 정의할 때 받는 변수
- **인수(Argument):** 호출할 때 넣는 실제 값
- **`return`:** 값을 반환하고 함수 종료 (재사용 가능) vs **`print`:** 출력만 하고 끝 (재사용 불가)

### **4. 기본 입출력**

**입력 (`input`)**

- **기본:** 문자열(`str`)로 입력받음
- **변환:**
    - `text.split()`: 공백 기준으로 잘라 리스트 생성
    - `map(int, input().split())`: 여러 숫자를 한 번에 변환

**출력 (`print`)**

- **옵션:** `sep=" "` (사이 문자), `end="\n"` (끝 문자)
- **포맷팅 (f-string):** `f"이름은 {name}입니다"` 형태로 변수 삽입

### **5. 클래스 (Class)**

**개념**

- **클래스:** 객체를 찍어내는 **틀** (붕어빵 틀)
- **인스턴스:** 틀로 찍어낸 **실체** (붕어빵)

**핵심 요소**

- **`__init__` (생성자):** 객체 생성 시 자동 호출되어 변수(속성)를 초기화
- **`self`:** **객체 자기 자신**. 메서드 정의 시 첫 번째 인자로 필수
- **메서드:** 클래스 내부에 정의된 함수

**주요 기능**

- **상속:** `class 자식(부모):` 형태로 부모의 기능을 물려받음
- **오버라이딩:** 부모의 메서드를 자식 클래스에서 **재정의(수정)**하여 사용
- **클래스 변수:** 모든 인스턴스가 **공유**하는 변수 (수정 시 `클래스명.변수`로 접근)
