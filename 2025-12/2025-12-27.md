### 변수와 자료형

# 변수 (Variable)

**1. 정의**

- 데이터를 저장하는 **메모리 공간에 붙인 이름**
- 복잡한 메모리 주소 대신, 쉬운 이름을 통해 데이터에 접근

**2. 핵심 기능: 동적 타이핑 (Dynamic Typing)**

- 변수를 선언할 때 자료형(Data Type)을 미리 명시할 필요 X
- 할당되는 값에 따라 Python 인터프리터가 **자동으로 자료형을 결정**

**3. 작명 규칙:**

- 영문, 숫자, 언더바(`_`) 사용 가능
- **숫자로 시작 불가** (ex: 1_number **X**)
- **대소문자 구분** (A, a는 다르게 인식)
- 예약어 사용 불가 (`if`, `class`, `for` 등)

```python
name = "Python"   # "Python"는 문자열이므로 str 타입으로 자동 결정
number = 25         # 25는 정수이므로 int 타입으로 자동 결정
```

- 요약
    - 변수는 '데이터를 담는 그릇'
    - 예시 → 단축번호
        - 값 (전화번호) : 010-1234-5678
        - 변수 (단축번호) : 1번
        - `num_1    =    "010-1234-5678”`
    - Python은 그릇에 담긴 내용물을 보고 **어떤 종류인지(자료형)'를 자동으로 결정**


# 자료형

### 1. 기본 자료형 (Scalar Types)

- 하나의 값을 표현하는 기초적인 데이터 단위

### (1) Integer (정수형, `int`)

소수점이 없는 숫자

- **표현:** `77`, `14`, `0` 등 숫자(**정수**)만 작성
- **특징:**
    - **크기 무제한:** 메모리가 허용하는 한, 아무리 큰 숫자(예: 99999999...)도 다룰 수 있습니다.
    - C언어나 자바처럼 `long` 타입을 따로 구분하지 않아도 됨
- **용도:** 개수, 순서, 인덱스 번호 등 딱 떨어지는 숫자 (정수)를 셀 때 사용

```python
num = 23
print(num)     
print(type(num)) 

num_num = 999999999999999999999999999999999999999999999999
print(num_num) 

count = 10
print(count + 10)
```

### (2) Float (실수형, `float`)

소수점이 포함된 숫자 (**실수**)

- **표현:** `3.14`, `0.01` 처럼 소수점(`.`)을 찍거나, `1.2e3` 등을 사용
- **특징:**
    - 소수점 필수**:** 정수 `1`과 실수 `1.0`은 다른 자료형으로 인식
    - 연산 결과에 소수점이 생기면 자동으로 float로 변환 (예: `3 / 9`).
- **용도:** 키, 몸무게, 환율, 원주율(`pi`) 등 정밀한 계산이 필요한 숫자 (실수) 데이터에 사용

```python
pi = 3.14
print(pi)    
print(type(pi))  

sci = 1.2e3
print(sci)     

div = 3 / 12
print(div)    
```

### (3) Boolean (논리형, `bool`)

오직 '참'과 '거짓' 두 가지 값만 존재하는 자료형

- **표현:** `True` (참), `False` (거짓)
- **특징:**
    - 숫자는 `0`, 문자열은 비어있는 것(`""`), 리스트는 빈 것(`[]`)을 `False`로 취급
    - 나머지는 모두 `True`입니다.
    - **연산:** `1 > 2` 같은 비교 연산의 결과로 튀어나옵니다.
- **용도:** `if` 문에서 조건을 판단할 때 사용

```python
is_active = True
is_admin = False
print(type(is_active)) 

result = (10 > 5)    
print(result)         

print(bool(1))    
print(bool(0))  
```

### (4) String (문자열, `str`)

글자들의 나열(집합)

- **표현:** 큰따옴표(`" "`)나 작은따옴표(`' '`)로 감싼 형태
- **특징:**
    - **변경 불가:** 한번 `"Hello"`라고 만들면, 그 안의 글자를 `'a'`로 바꿀 수 없음
        - ⇒ 수정하려면 아예 새로운 문자열을 생성해야 함
    - **시퀀스:** 글자마다 번호(인덱스)가 있어서 순서대로 꺼낼 수 있음 (`s[0]`, `s[1]`, `s[2]`, `s[3]` ---)
- **용도:** 이름, 메시지, 아이디 등 텍스트 데이터를 다룰 때 사용

```python
greeting = "Hello"
print(greeting)
print(type(greeting)) 

print(greeting[0])    
print(greeting[4])   

greeting = "hello" 
```

### (5) NoneType (`None`)

"값이 없음" 그 자체를 나타내는 자료형

- **표현:** `None`
- **특징:** `0`이나 `False`와는 다르게 `None`은 아예 아무것도 없는 상태
- **용도:** 변수 생성 후, 넣을 값이 없거나, 함수가 아무것도 반환하지 않을 때 사용

```python
data = None
print(data)          
print(type(data))     

val_zero = 0          
val_none = None      

print(val_zero == val_none)

result = None         
```

<기본 자료형>

| **자료형** | **설명** | **예시** | **특징** |
| --- | --- | --- | --- |
| **int** | 정수 | `a = 77`, `b = -14` | 딱 떨어지는 숫자 (정수) |
| **float** | 실수 | `f = 3.14`, `f2 = 1.2e3` | 소수점이 포함된 숫자 (실수) |
| **bool** | 논리 | `is_ok = True` | 참(`True`) / 거짓(`False`) |
| **str** | 문자열 | `s = "Hello"` | 문자의 집합. 따옴표(`"`, `'`) 사용. |
| **None** | 없음 | `data = None` | 값이 존재하지 않음을 표현 |

### 4. 컬렉션 자료형 (Collection Types)

- 여러 데이터를 한 번에 담는 그릇
    - **기호 :** 데이터를 **어떤 괄호**로 감싸느냐(`[]`, `()`, `{}`)에 따라 파이썬이 **서로 다른 자료형**으로 인식
    - **수정 여부 :** 데이터가 메모리에 생성되고, **그 안의 값 변경(추가, 삭제, 교체) 가능 여부**
    
    ### 1. List (리스트)
    
    - 리스트는 여러 값을 저장할 수 있으며, 순서가 있고 변경 가능
    - **기호:** `[ ]` (대괄호)
    - **변경** **가능 (O)**
        - 언제든지 내용을 지우거나, 새로운 내용을 덧붙일 수 있음
    - **특징:**
        - **순서:** 있음 (인덱스로 0번, 1번, 2번... 접근 가능)
        - **중복:** 허용 (같은 값이 여러 개 들어가도 됨)
    - **용도:** 데이터의 추가와 삭제가 자주 일어나는 코드에서 사용
    
    ```python
    scores = [80, 90, 90, 75] 
    print(f"원본: {scores}")
    
    print(f"첫 번째 점수: {scores[0]}") 
    
    scores.append(100)  
    scores[0] = 85      
    print(f"수정 후: {scores}") 
    ```
    
    ### 2. Tuple (튜플)
    
    한 번 쓰면 고칠 수 없는 리스트
    
    - **기호:** `( )` (소괄호)
    - **변경 불가능 (X)**
        - 한 번 생성하면 그 안의 값을 절대 바꿀 수 없음
    - **특징:**
        - **순서:** 있음 (리스트처럼 인덱스로 접근 가능)
        - **중복:** 허용
    - **용도:** 좌표나 설정값처럼 프로그램이 실행되는 동안 **변하면 안 되는 중요한 값**을 저장할 때 사용
    
    ```python
    coords = (37.5, 127.0)
    print(f"좌표: {coords}")
    
    print(f"위도: {coords[0]}") 
    ```
    
    ### 3. Dict (딕셔너리)
    
    단어와 뜻이 짝지어 있는 '사전'같은 형태
    
    - **기호:** `{ }` (중괄호) - **Key: Value** 형태
    - **변경** **가능 (O)**
        - 키(Key)를 이용해서 값(Value)을 언제든 수정할 수 있습니다.
    - **특징:**
        - **순서:** 없음 (인덱스 번호 대신 'Key'로 값을 찾음)
        - **중복:** **Key는 중복 불가**, Value는 중복 가능
    - **용도:** 회원 정보(이름: 파이썬, 나이: 22)처럼 이름표(Key)와 실제 데이터(Value)를 쌍으로 관리할 때 사용
    
    ```python
    profile = {"name": "Hong", "age": 25}
    print(f"프로필: {profile}")
    
    print(f"이름: {profile['name']}") 
    
    profile["age"] = 26     
    profile["job"] = "Dev"    
    print(f"수정 후: {profile}")
    
    ```
    
    ### 4. Set (집합)
    
    중복을 허락하지 않는 목록
    
    - **기호:** `{ }` (중괄호) - 값만 나열된 형태
    - **변경** **가능 (O)**
        - 값을 추가하거나 뺄 수 있음
    - **특징:**
        - **순서:** 없음 (순서가 없으므로 인덱싱 불가)
        - **중복:** **불가능** (똑같은 값을 넣으면 하나만 남고 다 사라짐)
    - **용도:** 데이터에서 **중복을 제거**하고 싶을 때나, 교집합/합집합 연산이 필요할 때 사용
    
    ```python
    nums = {1, 2, 3, 1, 1}
    print(f"중복 제거 결과: {nums}") 
    
    nums.add(4)       
    nums.remove(2)    
    print(f"최종 집합: {nums}")
    
    ```
    
    <컬렉션 자료형>
    

| **자료형** | 설명 | **기호** | **순서** | **중복** | **변경** | **특징** |
| --- | --- | --- | --- | --- | --- | --- |
| **List** (리스트) | 순서가 있는 변경 가능한 배열 | [ ] | O | O | **가능** | 데이터의 추가/삭제가 자주있는 코드 |
| **Tuple** (튜플) | 순서가 있는 변경 불가능한 배열 | ( ) | O | O | **불가능** | 변하면 안 되는 고정된 값 목록 |
| **Dict** (딕셔너리) | 키-값 쌍을 저장하는 자료형 | { } | X | Key X | **가능** | Key-Value 쌍으로 데이터 관리 |
| **Set** (집합) | 중복을 허용하지 않는 집합 | { } | X | X | **가능** | **중복 제거**나 교집합/합집합 계산 시 사용 |

### 5. 형 변환 (Type Casting)

- 자료형을 강제로 변경하는 방법
- **숫자 변환:** int(3.14) → 3, float(5) → 5.0.

```python
f_num = 3.14
print(f"int(3.14) 결과: {int(f_num)}")  
# 출력: 3

i_num = 5
print(f"float(5) 결과: {float(i_num)}") 
```

- **문자 변환:** str(123) → "123".

```python
num = 123
s_num = str(num)
print(f"변환 결과: {s_num}, 타입: {type(s_num)}")
print(f"이어붙이기: {s_num + '456'}") 

s_score = "100"
real_score = int(s_score)
print(f"숫자 연산 가능: {real_score + 50}") 
```

- **컬렉션 변환:** list((1, 2)) → [1, 2], set([1, 1, 2]) → {1, 2} (중복 제거됨).

```python
tup = (1, 2)
lst = list(tup)
lst.append(3) 
print(f"리스트 변환: {lst}")

dup_list = [1, 1, 2, 2, 3]
clean_set = set(dup_list)
print(f"중복 제거(Set변환): {clean_set}")
```
